Изменение

Изменение отчислений

Изменение отчисление делаеться через создание копии комиссии, тк их редактирование недоступно.

{% note %} Почему? Данные об отчислении за полученую заявку храняться в комиисии. Для того чтоб эти данные не терялись, редактирование отчислений в комиссии недоступно. {%end%}

Для создание копии комиссии, необходима нажать на соответствующую иконку (карандашик)

{%img%} https://i.imgur.com/nKQBQcN.png {%end%}

Далее необходимо внести актуальные новые данные по отчислениям.

{% notice %} Если изменилась лишь комиссия партнерки, новость писать не нужно. {% end %}
