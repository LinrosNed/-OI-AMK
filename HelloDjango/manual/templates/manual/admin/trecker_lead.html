Трекер Лиды

Назвачение

В трекере лидов отображаються все заявки(лиды) которые пришли в систему от вебов. В нем можно посмотреть различную информация по заявке. Трекер находиться со ссылке.

{%img%} https://i.imgur.com/mKpTP6e.png {%end%}

Фильтры

C помощью фильтров мы можен найти необходимые нам заявки, сортирую и фильтрую их по разным значениям

{%img%} https://i.imgur.com/FUXFp43.png {%end%}

Описание некоторых полей полей

Внизу у нас есть таблица с лидами и различными параметрами

{%img%} https://i.imgur.com/NkrH8yp.png {%end%}
  • Дата создания - дата создания заявки с системе КМА
  • Дата обобрения - дата ободрения заявки(если стоит прочерк - заявка не ободрена)
  • Номер заказа - номер заказа в системе КМА
  • Время пользователя - время создания заявки на стороне вебмастера
  • Оффер - оффер, на который отправлена заявка
  • Модель оплаты -
  • Рекламодатель - рекл, которому отправлена заявка
  • Кампания - рекл и гео, на которые ушла заявка
{%img%} https://i.imgur.com/QCjniAG.png {%end%}
  • Комиссия - комиссия по данной заявке
  • Общая комиссия - общая комиссия по данной заявке
  • Статус - статус завявки с системе КМА
  • Начисление - начисленая сумма по заявке
  • Балы - балы(свинкоины) за заявку
  • Вебмастер - имя и логие веба
  • RefID -
  • Поток - id потока из личного кабинета веба
  • IP - апи адрес отправителя заявки
  • Регион по IP - регеон отправителя, вычисленый по айпи
{%img%} https://i.imgur.com/sQ5dErZ.png {%end%}
  • Гео - Страна из которой пришла заявка
  • ФиО - имя оставленое заказчиком
  • Телефон - обработаный трекером телефон(убирает +)
  • Исходный телефон - телефон указаный заказчиком
  • Причина автофейка - елси заявка фэйкнулась, будет указана причина
  • Лэндинг и url - ленд и его линк (если прочерк - заявка отправлена по api)
  • Прелэндинг и url - преленд и его линк (если прочерк - заявка отправлена по api)
  • Устройство - тип устройства отправителя
  • Моб - признак мобильности
{%img%} https://i.imgur.com/ZLhsGV3.png {%end%}
  • Дата холда -
  • Внешний ID - адйди заявки на стороне рекламодателя(если прочерк - он использует айди КМА)
  • Браузер и версия - браузер и версия у пользователя
  • Операционная система - ОС пользователя
{%img%} https://i.imgur.com/mVM7OLG.png {%end%}

Причины фэйков
