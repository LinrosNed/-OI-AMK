Лэнд и прелэнд

Есть два типа сайтов - лэнд и прелэнд. Визуально они легко отличимы. Лэнд это сайт о продукте, он стилизован, похощ на стандартные однастоаничники. Прелэнды обычно выглядят как новостные сайты,медицинские порталы или блоги

Адаптивность

Адаптивность — одно из ключевых требований, предъявляемых к современным сайтам. Ресурс должен одинаково хорошо демонстрироваться на экране компьютера, планшете, смартфоне.

Локализация

На одном сайте может быть несколько вариантов переводов. Если они есть, сбоку будет присутствовать окошка с флагами стран, нажав на который можно попасть на нужный перевод.

{% img %} переход между языками$https://i.imgur.com/sYyBZ8j.png {%end%}

После перехода на выбраный язык у ссылки появиться utm-мета lang="xx", где вместо xx будет указан iso код выбраного языка.

{% code safe%} https://apolloss-new.com/?lang=bg {%end %}

Справа налево

В арабском языке текст идет справа налево, соотвественно весь текст на араском сайте должен идти справа налево (включаю инпуты, плэйсхолдеры, селекты и прочее).

Кэширование
